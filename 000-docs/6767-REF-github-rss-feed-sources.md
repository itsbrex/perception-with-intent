- **Practitioners:** Fast.ai, ML Mastery, Andrej Karpathy

- Browser blogs (Chrome, Firefox, Safari, Edge)
- JavaScript frameworks (React, Vue, Angular)
- CSS resources
- Developer tools
- Accessibility

### 6. Security Feeds (~100 feeds)
**Repo:** [cudeso/OPML-Security-Feeds](https://github.com/cudeso/OPML-Security-Feeds)
**OPML:** `feedly.opml`, `fever-security.opml`

Sources:

- Packet Storm, ZDI

### 7. Cybersecurity RSS (~200 feeds)
**Repo:** [zer0yu/CyberSecurityRSS](https://github.com/zer0yu/CyberSecurityRSS)
**OPML:** `tiny.opml`, `CyberSecurityRSS.xml`

- Bug bounty
